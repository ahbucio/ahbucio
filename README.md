# Adnan Bucio

Operator-entrepreneur in Guadalajara. Roughly ten years in operations and finance — CFO at a women's apparel manufacturer, a cannabis formulation startup, and since 2023 my own product, GDL XP.

I came into engineering through a product, not a bootcamp. When no-code stopped being enough for what GDL XP needed and hiring a developer was too slow, I picked up real tooling. The pinned repo is the result — a Node.js + TypeScript pipeline that uses Playwright to scrape Guadalajara event venues (starting with Auditorio Telmex), runs the unstructured content through Gemini 2.5 Flash for structured LLM extraction, and persists to Postgres. Live on Railway, end-to-end.

This GitHub account is new because I never needed one before. The product it's powering isn't.

Stack: Node 20, TypeScript strict, Express 5, Drizzle ORM + Neon Postgres, Zod on every API boundary, Vitest + supertest, @google/genai SDK, Playwright. Auto-deployed on Railway.

Currently: building toward Bubble integration for GDL XP, then containerization and cloud (Docker, Cloud Run).

Based in Zapopan, Jalisco. English fluent (EF SET C2).
